PLVR WP FAQ provides an easy way to add FAQ items to your website.

== Description ==

Adds FAQ System on any WordPress powered site. The plugin is based on custom post type to handle FAQ items. The plugin comes with FAQ category support.

###Shortocode
`[plvr_faq]`


### Plugin Features
*   Custom Post-Type driven
*   Category based FAQ control
*   Custom category order
*   Support unlimited FAQs
*   Support border
*   Styled Custom title
*   Can be displayed all, or in FAQ groups by using the "category" taxonomy

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plvr-wp-faq` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
